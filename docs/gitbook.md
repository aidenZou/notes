# gitbook

[https://www.gitbook.com](https://www.gitbook.com)

```shell
# 安装
npm install -g gitbook-cli

mkdir book
cd book
# 初始化书籍目录
gitbook init
# 编译和预览书籍
gitbook serve
```

用浏览器打开 http://127.0.0.1:4000 查看书籍的效果
