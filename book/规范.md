## Vue

比较重要的规范包括(其他在可以在文档发布后查看)

- 事件命名前缀 on-
- 组件命名：若与原生同名，使用 x-*, 若有子组件，使用*-item
- css 前缀 vux-

另外还有发布节奏的确定：
	•	每周三发布一个小版本。除周三外发布的都是rc版本。
